# A Lo Cubano Boulder Fest - Setup Complete! 🎉

## 🚀 Quick Start

1. **Start the server:**
   ```bash
   ./start.sh
   # or
   python3 server.py
   ```

2. **Open your browser:**
   Navigate to: http://localhost:8000

3. **Explore the designs:**
   - Click on any design card on the landing page
   - Navigate through each design's pages
